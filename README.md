## installation
not windows
```
git clone --recurse-submodules https://github.com/mintoya/my-nvim/ ~/.config/nvim && nvim
```
windows powershell
```
git clone --recurse-submodules https://github.com/mintoya/my-nvim/ $env:USERPROFILE\AppData\Local\nvim
nvim
```
